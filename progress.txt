## Итерация 3 — 2026-03-01

**Статус:** Завершено
**Что сделано:**
- Реализован `src/reviewmind/core/llm.py` — async wrapper вокруг `openai.AsyncOpenAI`
- Класс `LLMClient` с параметрами из config: api_key, base_url, model, temperature, max_tokens
- Метод `generate(system_prompt, user_message, temperature?, max_tokens?, model?, messages?) → str`
- Поддержка multi-turn контекста через параметр `messages` (chat history)
- Retry с exponential backoff (1s, 2s, 4s) для transient ошибок: RateLimitError, APITimeoutError, APIConnectionError, InternalServerError
- Non-retryable ошибки (AuthenticationError, BadRequestError) сразу выбрасывают `LLMError`
- Переключение GitHub Models ↔ OpenAI через замену `base_url`
- Async context manager support (`async with LLMClient() as client`)
- Custom exception `LLMError` для всех ошибок LLM
- 35 unit-тестов в `tests/unit/test_core/test_llm.py`: init (8), message building (4), generate (7), error handling (10), retry timing (1), context manager (2), LLMError (3)

## Итерация 4 — 2026-03-01
**Фича:** TASK-005 — FastAPI приложение: app factory, lifespan, /health endpoint, structlog логирование
**Статус:** Завершено
**Что сделано:**
- Реализован `src/reviewmind/main.py` с FastAPI app factory (`create_app`), asynccontextmanager lifespan
